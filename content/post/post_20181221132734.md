## HugoでMermaid.jsを使えるようにした

こう書けば、  

```
｛｛< mermaid align="left" >｝｝
  graph LR;
  A[Hard edge] -->|Link text| B(Round edge)
  B --> C{Decision}
  C -->|One| D[Result one]
  C -->|Two| E[Result two]
｛｛< /mermaid >｝｝
```

こうなる。  

{{< mermaid align="left" >}}
  graph LR;
  A[Hard edge] -->|Link text| B(Round edge)
  B --> C{Decision}
  C -->|One| D[Result one]
  C -->|Two| E[Result two]
{{< /mermaid >}}

※ コードブロックでエラーが出るので安直に半角の'{', '}'を全角の'｛', '｝'に置換して回避してます…
